# まちるだいすけと めむろたんけん

小学校3・4年生が、地域学習教材「めむろ学」をもとに芽室町を楽しく学ぶ、クイズWebアプリです。
サーバーも外部サービスも使わない「静的サイト」なので、無料で長期間運用できます。

## 特長

- 3コース・各25問(合計75問)、選択肢は毎問シャッフル
- 回答後は「つぎへ」ボタンで、自分のペースで解説を読んでから進める
- 25問全問正解で認定証を表示、まちがえた問題だけの復習機能
- 3年生までの漢字だけを使用し、漢字にはふりがなを表示
- 途中経過は端末内(ブラウザ)にのみ保存。外部API・ログイン・個人情報の保存なし
- Chromebook、スマートフォン、キーボード操作、スクリーンリーダーに対応

## ファイル構成(どこを触ればいいか)

```
index.html        画面の骨組み。ふだん触りません
css/style.css     見た目(色・大きさ)。色を変えたいときはここ
js/data.js        ★問題データ。問題の追加・修正はここだけでOK
js/furigana.js    ふりがな辞書。新しい漢字の文を足したときだけ触る
js/app.js         動きの仕組み。ふだん触りません
images/           問題に使う写真
teacher-materials/ 先生用の問題一覧PDF(児童画面からはリンクされません)
IMAGE-SOURCES.md  写真の出典一覧
```

## 問題を直す・増やす(js/data.js)

`js/data.js` の `questions` の中に、1問ずつこの形で並んでいます。

```js
{ level:"beginner",                  // コース: beginner / intermediate / advanced
  question:"芽室町の代表的な作物は？",
  image:"corn.jpg",                  // imagesフォルダの写真名。""なら写真なし
  choices:["スイートコーン","パイナップル","バナナ"],  // 正解を必ず先頭に
  answer:0,                          // そのまま0でOK(表示時に自動シャッフル)
  explanation:"スイートコーンは芽室町を代表する作物の一つです。",
  source:"めむろ学 p.26–31" },
```

守ってほしいこと:

1. 各コースは必ず25問にする
2. 正解は choices の先頭に書く(answer は 0 のまま)
3. 公開前に「めむろ学」最新版と照合し、source のページを確認する
4. 新しい文に4年生以上の漢字が入るときは、`js/furigana.js` にも読みがなを追記する

## ふりがなを追加する(js/furigana.js)

`kidTextMap` は「元の文」→「ふりがな付きの表示」の対応表です。

```js
"芽室町の新しい問題文？": "めむろ<ruby>町<rt>まち</rt></ruby>の<ruby>新しい<rt>あたらしい</rt></ruby>もんだいぶん？",
```

注意: 左側は data.js に書いた文と一字一句同じにしてください。1文字でも違うと、ふりがなが付かずそのまま表示されます。追記したら『ふりがな監修チェックリスト.xlsx』にも足して、読みの監修を受けてください。

## 写真の追加(images/)

写真を `images` フォルダへ入れ、`js/data.js` の `image` にファイル名を書きます。指定した写真がまだ無い場合は自動で `placeholder.jpg` が表示されます。児童・施設が写る写真は利用許可と肖像権を、教材の写真は著作権・利用条件を必ず確認してください。

## GitHub Pagesで公開・更新する

1. このフォルダの中身をGitHubリポジトリのルートへアップロード
2. Settings → Pages → 「Deploy from a branch」で `main` / `/ (root)` を指定
3. 表示されたURLをChromeで開く

ファイルを更新したときは、`index.html` の中の `?v=20260704` という数字を今日の日付に変えると、児童の端末に古い表示が残る問題(キャッシュ)を防げます。

## 保存データについて

途中経過だけをブラウザの Local Storage(`memuro-tanken-progress-v1`)へ保存します。氏名などの個人情報は保存しません。「もう一度挑戦」で前の途中記録は消えます。
